# Table Cells

`@synerise/ds-table-new` exports pre-built cell components for rendering common content patterns inside table columns. Import each cell directly by name.

- [ActionCell](#actioncell) — flex container for action buttons
- [AvatarLabelCell](#avatarlabelcell) — avatar with title and labels
- [CopyableCell](#copyablecell) — text with copy-to-clipboard
- [EditableCell](#editablecell) — inline editable text
- [FlagLabelCell](#flaglabelcell) — country flag with label
- [IconLabelCell](#iconlabelcell) — icon with text label
- [IconTooltipCell](#icontooltipcell) — icon, label, and tooltip
- [InputNumberCell](#inputnumbercell) — numeric input
- [LabelsWithShowMore](#labelswithshowmore) — labels with "+N more" modal
- [StarCell](#starcell) — star/favorite toggle
- [StatusLabelCell](#statuslabelcell) — status badge with label
- [TagIconCell](#tagiconcell) — tag with icon
- [TagsGroupCell](#tagsgroupcell) — group of tags

```tsx
import { IconLabelCell, type ColumnDef } from '@synerise/ds-table-new';

const columns: ColumnDef<MyData, unknown>[] = [
  {
    accessorKey: 'name',
    header: 'Name',
    cell: (info) => (
      <IconLabelCell label={info.getValue() as string} />
    ),
  },
];
```
