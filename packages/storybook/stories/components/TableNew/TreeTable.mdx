# TreeTable

Renders hierarchical data with visual indentation, colored vertical lines, and expand/collapse controls. Built on top of `Table` using TanStack's expanding row model -- not a separate table implementation.

## Quick overview

- Data must have a `children` property (configurable via `childrenColumnName`)
- Indentation: 42px per level, 8-color cycling vertical lines (yellow, green, cyan, violet, orange, purple, blue, grey)
- Parent rows show an expand/collapse button; leaf rows show a connector icon
- Supports uncontrolled (internal state) and controlled (`expandedRowKeys`) expand modes
- All `Table` props are supported (search, selection, pagination, sorting, etc.)

## Basic example

Click the expand buttons to reveal child rows. Indentation and colored lines are rendered automatically based on row depth.

```tsx
import { TreeTable } from '@synerise/ds-table-new';

const data = [
  {
    key: 'analytics',
    name: 'Analytics',
    children: [
      { key: 'dashboards', name: 'Dashboards' },
      { key: 'funnels', name: 'Funnels' },
    ],
  },
];

const columns = [
  { header: 'Name', accessorKey: 'name', id: 'name' },
];

<TreeTable data={data} columns={columns} rowKey="key" />
```

## All rows expanded

Pass `defaultExpandAllRows` to start with every row expanded. The tree will compute all row keys from the data and expand them on mount.

```tsx
<TreeTable
  data={data}
  columns={columns}
  rowKey="key"
  defaultExpandAllRows
/>
```

## Controlled expand state

Manage `expandedRowKeys` externally for full control over which rows are expanded. Use `onExpandRow` to react to expand/collapse events.

```tsx
const [expandedKeys, setExpandedKeys] = useState(['analytics']);

const handleExpandRow = (key, expanded) => {
  setExpandedKeys(prev =>
    expanded ? [...prev, key] : prev.filter(k => k !== key)
  );
};

<TreeTable
  data={data}
  columns={columns}
  rowKey="key"
  expandedRowKeys={expandedKeys}
  onExpandRow={handleExpandRow}
/>
```

## Indentation system

Each indent level is 42px wide with a 2px colored vertical line on the left. Colors cycle through 8 values:

The "active" bar is the deepest indent level (closest to the row content). All other bars use the inactive hue. Colors cycle back to yellow after grey for trees deeper than 8 levels.

## Data structure

Row data must include a `children` property (or the property specified by `childrenColumnName`) containing an array of child rows with the same shape:

```typescript
type TreeRow = {
  key: string;
  name: string;
  children?: TreeRow[];
};
```

The tree depth is determined automatically by TanStack React Table's expanding row model. There is no hardcoded depth limit.
