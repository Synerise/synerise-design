# RichText

A WYSIWYG rich text editor built on Tiptap. It edits HTML, Markdown, or a portable
JSON block model (`RichTextDocument`) behind a single controlled `value` / `onChange`
API, with a configurable formatting toolbar.

## Quick overview

- Import: `import RichText from '@synerise/ds-rich-text';`
- Must render inside `DSProvider` from `@synerise/ds-core` (theme + i18n — the code
  snippet copy button uses `react-intl`).
- Controlled (`value` + `onChange`) or uncontrolled (`defaultValue`).
- `format` selects the value type: `'html'` (default) · `'markdown'` · `'document'`
  (JSON block model that round-trips to `@synerise/ds-rich-text-renderer`).
- The toolbar is fully configurable via `toolbarFeatures`; hide it entirely with
  `hideToolbar`.
- `subtle` turns the editor into a SubtleForm-style inline edit: borderless preview,
  grey hover affordance with an edit icon, click to edit, click outside to go back.
  Hover/blur transitions are opt-in via `animations`.
- Links typed without a protocol (`www.google.com`) are normalized to
  `https://www.google.com` and open in a new tab.

## Edit with AI

Providing `onEditWithAI` adds an "Edit with AI" toolbar button. The handler receives
the current selection (or the full text when nothing is selected) and its resolved
string replaces it. The design system performs no AI call itself.

## Notes & tips

- **Persist `format="document"`** when the content will be displayed with
  `RichTextRenderer` — it avoids HTML parsing at render time and keeps the payload
  portable.
- Links open in a new tab (`target="_blank"` + `rel="noopener noreferrer"`); URLs
  entered without a protocol are prefixed with `https://` automatically.
- The single-line **code snippet** block (`codeSnippet` feature) serializes as
  `<pre data-type="code-snippet">` so it round-trips without colliding with regular
  code blocks, and shows a copy button (via `@synerise/ds-copy-icon`).
- In `subtle` mode the toolbar mounts only in edit mode; with `animations` enabled the
  preview→edit transitions reuse the SubtleForm hover/blur animations, otherwise
  hovering the preview only tints the background.
- All UI strings are overridable via `texts` — pass translated strings from
  `react-intl` in the consuming app.
